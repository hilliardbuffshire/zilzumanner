🔗 친구에게 공유하기

이 HTML 파일을 친구들과 공유하는 방법은 두 가지입니다.
아래 중 하나를 선택하세요.
방법 1 — 파일 직접 공유 (가장 간단)
1
이 HTML 파일(whale_dashboard_v4.html)을 카카오톡 / 디스코드 / 이메일로 친구에게 전송
2
친구가 파일을 다운받아 브라우저에서 열면 끝. 별도 서버 불필요.
방법 2 — GitHub Pages (링크 공유)
1
github.com 가입 → 새 Repository 생성 (이름 아무거나)
2
이 HTML 파일을 index.html로 이름 바꿔서 업로드
3
Repository Settings → Pages → Source: main branch → Save
4
https://유저명.github.io/레포이름 링크를 친구들에게 공유. 완전 무료 / 24시간 접속 가능.
방법 3 — Google Sites (구글 계정만 있으면 OK)
1
sites.google.com 접속 → 새 사이트 만들기
2
페이지 추가 → HTML 삽입(Embed) → 이 파일 내용 붙여넣기 → 게시
3
생성된 구글 사이트 링크를 친구들과 공유
